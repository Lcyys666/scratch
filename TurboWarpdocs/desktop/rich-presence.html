丰富状态显示

TurboWarp Desktop 支持丰富状态显示功能，可以在某个聊天应用中显示您正在处理的项目名称和已处理时长。

由于 Scratch 指南的限制，我们不确定是否可以在这里提及这个功能支持的具体应用名称，所以我们将其称为"聊天应用"。

启用丰富状态显示​

为了保护隐私，丰富状态显示默认是禁用的。要启用它，请通过左上角的设置菜单打开桌面设置：

如果桌面设置按钮不存在，说明您使用的是旧版本；请更新到 1.12.0 或更高版本。在打开的窗口中，勾选"启用丰富状态显示"选项。如果该选项不可用，请查看本页面底部的支持的平台。

您可能还需要在聊天应用中更改一些设置：

  • 启用用户设置 > 活动隐私 > 与他人分享您的检测到的活动。
  • 对于每个您想要分享状态的服务器，右键点击服务器图标，然后启用隐私设置 > 活动状态。

您的状态可能需要最多 15 秒才会出现、更新或消失。如果仍然没有出现，请查看下面的支持的平台。

支持的平台​

Windows​

在 Windows 上，无论您如何安装 TurboWarp Desktop 或聊天应用，丰富状态显示都能正常工作。

macOS​

warning

如果 TurboWarp Desktop 是从 Mac App Store 安装的，丰富状态显示将无法工作。

Linux​

为了获得最佳效果，请使用 .deb 包、Debian 仓库、Arch 用户仓库、AppImage 或 tarball 将 TurboWarp Desktop 安装为原生应用。作为 Flatpak 安装的 TurboWarp Desktop 可能需要手动权限覆盖。作为 Snap 安装的 TurboWarp Desktop 完全无法工作。详见下表：

聊天应用原生版聊天应用 Flatpak聊天应用 Snap
TurboWarp Desktop 原生版✅✅✅
TurboWarp Desktop Flatpak见下方命令✅见下方命令
TurboWarp Desktop Snap❌❌❌

对于 Flatpak 版本的 TurboWarp Desktop，您可能需要在终端中运行以下命令并重启 TurboWarp Desktop 才能使丰富状态显示功能正常工作：

# 对于原生版聊天应用
flatpak override org.turbowarp.TurboWarp --user --filesystem=xdg-run/$(printf "\x64\x69\x73\x63\x6f\x72\x64")-ipc-{0..9}
# 对于 Snap 版聊天应用
flatpak override org.turbowarp.TurboWarp --user --filesystem=xdg-run/snap.$(printf "\x64\x69\x73\x63\x6f\x72\x64"):create

还有一个小注意事项：如果 TurboWarp Desktop 在聊天应用之前启动，丰富状态显示可能无法工作。

这些权限可以通过以下命令撤销：

flatpak override org.turbowarp.TurboWarp --user --reset

聊天应用修改版​

如果您使用修改版或第三方版本的聊天应用，请参考该修改版的文档了解如何启用 RPC。

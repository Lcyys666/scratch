未共享的项目不再可见

由于 Scratch API 的更改，未共享的项目不能再在 TurboWarp、forkphorus 和其他第三方网站上打开。

本页回答了人们的大多数问题，并列出了一些解决方法。在与其他讨论这些更改之前，请完整阅读本页，以避免传播错误信息。

warning

除了 scratch.mit.edu 之外的任何要求您输入 Scratch 密码的网站都是骗局，即使它声称可以让您与其他用户共享未共享的项目。您的账户将被盗用，项目将被删除。这条规则没有例外。

发生了什么​

我们要明确一点：这些更改是由 Scratch 团队做出的。TurboWarp 是一个与 Scratch 团队无关的第三方网站；我们没有做出这些更改。

从 Scratch API 下载项目现在需要一个"项目令牌"，对于未共享的项目，只有项目所有者才能访问该令牌。即使您在同一个浏览器中登录了您的 Scratch 账户，TurboWarp 也无法访问它。这些令牌是临时的，几分钟后就会过期，所以所有者不能只提供一次令牌就让它永远可见。

未共享的项目一直都是碰巧能用的功能，而不是 TurboWarp 的主要用途。像编译器和插件这样的功能一直是重点，并将继续适用于共享项目、从文件加载的项目和桌面应用程序。

解决方法​

用于测试您自己的项目： 您可以使用 Scratch 编辑器中的"文件 > 保存到您的计算机"和"文件 > 从您的计算机加载"菜单，在 TurboWarp 中加载您未共享的 Scratch 项目，或将 TurboWarp 中制作的项目上传到 Scratch。或者，许多人成功地在 TurboWarp 中主要使用网站或桌面应用程序处理他们的项目，并在完成时将项目上传到 Scratch（请记住在这样做时进行例行备份）。

用于协作： 与他人共享项目的最佳方式是简单地在 Scratch 网站上共享它。Scratch 社区非常友好。这就是 Scratch 希望您做的。共享未完成的项目是可以的。Scratch 已经 15 岁了，而 TurboWarp 只有两岁。在没有 TurboWarp 的 13 年里，协作进行得很好，并将继续顺利进行。

用于嵌入其他网站： 要在其他网站中嵌入未共享的项目，要么在 Scratch 上共享项目，要么使用 Scratch 编辑器中的"文件 > 保存到您的计算机"菜单将项目下载到您的计算机，然后使用 TurboWarp Packager 将该项目转换为可以嵌入的独立文件。

这是一件好事​

保护未共享的项目已经迟到了十年。

不要假装没有人因为不知道未共享的项目实际上并不私密而项目被盗，尽管 Scratch 网站说"只有您能看到它"。许多未共享的项目包含儿童、他们的朋友、他们的家人和其他个人信息的图片和视频，这是基于未共享的项目实际上是私密的假设。

在大多数其他大型网站中，"未共享"或"私密"的内容实际上是公开的，这将被视为严重的安全漏洞，通常有资格获得大量漏洞赏金。例如，YouTube 向一名安全研究人员支付了 5000 美元，因为他报告了一个允许他们查看任何私密视频的低分辨率图像的漏洞。

我们一直持这样的立场：如果人们希望未共享的项目真正私密，他们应该与 Scratch 团队交谈。也许有足够多的人这样做，让 Scratch 团队听取了意见。

对于开发者​

本节适用于开发第三方 Scratch 相关工具的人。

下载项目的新程序是首先从 https://api.scratch.mit.edu/projects/ID 获取 "project_token" 字段，然后使用它生成 URL https://projects.scratch.mit.edu/ID?token=TOKEN

如果您使用 JavaScript，这里有一些示例代码可以帮助您入门，这些代码可以在网页浏览器中运行。如果您的代码在服务器端运行（例如 Node.js），您应该将 https://trampoline.turbowarp.org/api/projects/ 替换为 https://api.scratch.mit.edu/projects/，因为服务器不受 CORS 的影响。我们不保证 trampoline.turbowarp.org 的在线时间；使用风险自负。您可能还对 sb-downloader（包含简单的 API）感兴趣，这是一个完整的项目下载器。

const getProjectMetadata = async (projectId) => {
    // 如果在网页浏览器中，您需要使用像 trampoline.turbowarp.org 这样的服务来访问 Scratch API。
    // 如果在 Node.js 中，您应该直接使用 https://api.scratch.mit.edu/projects/${projectId}。
    const response = await fetch(`https://trampoline.turbowarp.org/api/projects/${projectId}`);
    if (response.status === 404) {
        throw new Error('项目未共享或不存在');
    }
    if (!response.ok) {
        throw new Error(`获取项目元数据时发生 HTTP 错误 ${response.status}`);
    }
    const json = await response.json();
    return json;
};

const getProjectData = async (projectId) => {
    const metadata = await getProjectMetadata(projectId);
    const token = metadata.project_token;
    const response = await fetch(`https://projects.scratch.mit.edu/${projectId}?token=${token}`);
    if (!response.ok) {
        throw new Error(`获取项目数据时发生 HTTP 错误 ${response.status}`);
    }
    const data = await response.arrayBuffer();
    return data;
};

getProjectData('60917032').then((data) => {
    console.log(data);
}).catch((error) => {
    console.error(error);
});
